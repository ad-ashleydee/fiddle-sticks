This document will be a running list of all the stylistic things I learn along the way.


  1. Text Colour

    2. To change the colour of the text, before closing the tag that dictates the level (paragraph, header, etc.) you type: "style", then the equals sign, then - inside quotation marks - you type: "color", then colon, then the colour you'd like.

  2. Creating a Numbered List

    3. To create a list, you use the tag 'ol', which stands for 'ordered list'. Then you write the text for each point on the list inside individual 'li' tags. When you're finished, you close the 'ol' tag.

    To write underneath the titles in this way, you simply use the 'p' tag as usual, there's no need to use the line break tag.

    To create sub-clauses you use the 'ul' tag.

    • you then use the 'li' tag like normal.
    • I found it necessary to add a line break tag after the last 'li' sub-clause.

  3. Inserting a Photograph

    4. To insert a photo you open the 'img' tag, NB: you don't add the closing right angle bracket.

    You then type 'src =' and paste the image address.

    You then use 'width = "number of pixels of your choosing"' to set the size of the photo, and 'alt = "something" ' to add the alternative text.

    Technically you could add the closing right angle bracket here, but it's surplus to requirement. You simply close the '/img' tag.
